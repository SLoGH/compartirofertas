README v1.0 For WP User Frontend Version: 1.1 fork: 2RRR 4.0 alpha Showtime Example
====================================================================================

This is an example of the usage of WP User Frontend hooks for the 2RRR 4.0 fork.

A custom taxonomy "Showtime-120" is used as a post category and new posts are added using the follow example link

http://test.com/new-post?showtime_taxonomy=Showtime-120

"new-post" is a standard WordPress Page containing the shortcode  [wpuf_addpost]
